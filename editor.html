Generador de Notas

Guía rápida para el Cuerpo de la Nota:
Escribí normal. Separás los bloques con dos ENTER y usás estos prefijos:
  • ## Subtítulo para un subtítulo principal de sección.
  • ### Subtítulo para un subtítulo secundario (itálica).
  • > Texto de la cita para insertar una cita destacada con borde naranja.
  • --- (tres guiones) para una línea divisoria gris simple.
  • ~~~ (tres virgulillas) para insertar el hilo decorativo naranja ondeado.
Abre una pestaña nueva con el diseño final de la nota para revisión.
Genera el paquete de archivos estructurado listo para publicación.
