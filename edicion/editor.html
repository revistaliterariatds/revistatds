Generador de Notas

Guía rápida para el Cuerpo de la Nota:
Escribí normal. Para aplicar formatos del formato original separá los bloques con dos ENTER y usá estos prefijos:
  • ## Subtítulo para un subtítulo principal de sección.
  • ### Subtítulo para un subtítulo secundario (itálica).
  • > Texto de la cita para insertar una cita destacada con borde naranja.
  • --- (tres guiones) para una línea divisoria gris simple.
  • ~~~ (tres tildes de virgulilla) para insertar el hilo decorativo naranja ondeado.
Permite generar una vista previa completa de la nota en una pestaña nueva para revisar su diseño.
Genera el paquete de archivos estructurado listo para enviar al editor para su revisión y publicación definitiva.
